# SoleSense – Smart Insole Prototype

A hackathon-built prototype exploring wearable stress detection using foot-based sensing and biofeedback.

---

## Overview

SoleSense is a proof-of-concept system designed to detect stress-related behaviours through plantar pressure and motion patterns, and respond with targeted feedback. The project was developed within a limited timeframe as part of a hackathon, focusing on rapid prototyping and system integration.

---

## Concept

The idea is based on the observation that stress often manifests through unconscious physical behaviours such as:

* foot tapping
* shifting weight
* increased movement patterns

By capturing these signals through sensors embedded in an insole, the system aims to infer stress levels and provide responsive feedback.

---

## System Components

The prototype combines multiple sensing inputs:

* **FSR sensors** – detect pressure distribution and tapping patterns
* **IMU (motion sensor)** – captures movement and oscillations
* **Heart rate sensor (PPG)** – estimates physiological response

These inputs are used to form a simple stress indicator based on combined behavioural signals.

---

## Feedback Mechanism

The system explores reflexology-inspired actuation by targeting specific areas of the foot:

* Big toe region
* Midfoot region

These areas were selected based on conceptual links to stress-related physiological pathways.

---

## Output Interface

A lightweight interface was developed to:

* display live sensor readings
* visualise stress-related activity
* provide user feedback in real time

## Limitations

As a hackathon prototype, the system has several limitations:

* simplified stress detection model (non-clinical)
* limited sensor calibration
* basic actuation implementation
* no long-term validation

---

## Notes

This project was built under time constraints and is intended as an exploration of wearable sensing and biofeedback rather than a finished product.
